Promeet is a browser-based real-time video conferencing application built using JavaScript.
The project focuses on implementing peer-to-peer communication, real-time signaling, and asynchronous event handling for low-latency audio and video calls.

This project was built to understand the fundamentals of real-time communication systems and event-driven web applications.
